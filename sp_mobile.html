Smart Presensee (Mobile)

Mobile attendance system implementation with face recognition and real-time connectivity, integrated with automated email notifications to parents.

❮ ❯

Project Overview

Smart Presensee Mobile is a robust Flutter-based application designed for efficient and reliable attendance tracking. It leverages Face Recognition (via Google ML Kit) to ensure high integrity in attendance logging and integrates Gmail SMTP for automated email notifications to parents.

🎯 Challenge

Conventional attendance methods are prone to data loss and difficulties in generating accurate attendance recaps, making records inefficient and unreliable.

💡 Implementation

Integrated a face recognition-based attendance system to validate users during check-in and implemented Gmail SMTP services to automatically send attendance notifications to parents, ensuring transparency and timely communication.

Key Implementation Features

👤

Face Recognition

High-accuracy landmark-based identification using Google ML Kit for secure biometric authentication.

⚡

Real-time Sync

Firebase integration for instantaneous data updates across the entire ecosystem.

🔔

Email Notifications

Automated attendance alerts sent to parents via Gmail SMTP integration to ensure transparency and timely communication.

📲

Forgot Password with OTP

Secure password recovery using One-Time Password (OTP) sent via WhatsApp through Fonnte integration to verify user identity before resetting credentials.
